# Anita Javadi - Architect Portfolio Website

A modern, responsive portfolio website for Anita Javadi, showcasing her architectural work and professional skills.

## Features

### 🎨 Design & UI

- Modern, clean design with professional aesthetic
- Responsive layout that works on all devices
- Smooth animations and transitions
- Interactive project galleries
- Mobile-first approach

### 📱 Responsive Design

- **Desktop**: Full-width layout with grid systems
- **Tablet**: Optimized for medium screens (768px and below)
- **Mobile**: Mobile-first design for screens 480px and below
- Flexible navigation with hamburger menu on mobile

### 🏗️ Architecture Projects Showcased

1. **Apadana Hospital** - Healthcare facility design
2. **Residential Complex** - Modern residential development
3. **Girls Dormitory** - Educational residential facility
4. **Modern Cafe** - Commercial café design
5. **Facade Modeling** - Building facade design

### 🛠️ Technical Features

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive features and smooth scrolling
- **Font Awesome**: Professional icons
- **Google Fonts**: Inter font family for modern typography

### 📋 Sections

- **Hero Section**: Introduction with call-to-action buttons
- **About**: Professional background and statistics
- **Skills**: Technical skills organized by categories
- **Projects**: Filterable project gallery with detailed modals
- **Contact**: Contact form and social media links

## Usage

1. Open `index.html` in a web browser
2. Navigate through different sections using the navigation menu
3. Filter projects by category
4. Click on project cards to view detailed information
5. Use the contact form to get in touch

## Customization

### Colors

The website uses a professional color scheme:

- Primary: #e74c3c (Red)
- Secondary: #2c3e50 (Dark Blue)
- Accent: #667eea (Purple)
- Background: #f8f9fa (Light Gray)

### Typography

- Font Family: Inter (Google Fonts)
- Weights: 300, 400, 500, 600, 700

### Adding New Projects

1. Add project images to the appropriate folder in `Projects/`
2. Update the HTML in the projects section
3. Add corresponding modal content in `script.js`
